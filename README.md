# Agent Skills Repository

Collection of AI agent skills. Install skills from [skills.sh](https://skills.sh/) using:

```bash
npx skills add <owner/repo>
```

## Installing Skills from This Repository

```bash
# Azure DevOps integration
npx skills add shkreios/agents/skills/azure-devops

# Cursor agent semantic code search
npx skills add shkreios/agents/skills/cursor-agent

# Skill creation guide
npx skills add shkreios/agents/skills/skill-creator
```
